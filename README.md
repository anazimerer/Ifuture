## O que funciona

- Splashscreen
- Login, Signup e Cadastro de endereço
- Feed dos restaurantes
- Busca
- Página do restaurante
- Carrinho
- Perfil com ordens anteriores
- Edição de perfil e endereço
- Barra de Pedido em andamento

## O que não funciona

- Barra de categorias
